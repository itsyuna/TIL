# 문제

두 정수 a, b가 주어졌을 때 a와 b 사이에 속한 모든 정수의 합을 리턴하는 함수, solution을 완성하세요.
예를 들어 a = 3, b = 5인 경우, 3 + 4 + 5 = 12이므로 12를 리턴합니다.

<br/>

🔴 제한 조건

a와 b가 같은 경우는 둘 중 아무 수나 리턴하세요.

a와 b는 -10,000,000 이상 10,000,000 이하인 정수입니다.

a와 b의 대소관계는 정해져있지 않습니다.

<hr>

<br/>

💡 구현 생각

1. 지금 경우의 수가 2가지, a > b 또는 a < b.
근데 a와 b의 대소관계는 정해져 있지 않다고 했으니까,
2가지 경우의 수 모두 코드를 작성해준다.

2. a < b일때는, i = a; i <= b; i++ 로 a를 시작값으로 설정.
a > b일때는, i = b; i <= a; i++ 로 b를 시작값으로 설정.
나머지 경우인, a와 b가 같은 경우에는 둘 중 아무수나 리턴하라고 했으니까 a 리턴!

<br/>

🔻 내 풀이 방법

function solution(a, b) {
    var answer = 0;
   
    // a가 b보다 작을 경우. Ex) 3 < 5
    if (a < b) {
        for ( let i = a; i <= b; i++) {
            answer += i;
        }
    // a가 b보다 클 경우. Ex) 5 > 3
    } else if (a > b) {
        for ( let i = b; i <= a; i++) {
            answer += i;
        }
    // a와 b가 같을 경우, 둘 중 a로 리턴.
    } else {
        answer += a;
    }
    
    return answer;
}
