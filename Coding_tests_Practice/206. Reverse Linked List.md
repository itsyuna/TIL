# Leetcode 문제

### 💡 구현 생각

<br>

current와 prev를 마들어주고 각각 next로 담아준다.

그리고 나서 prev를 반환해준다.

<hr>

🔻 코드

```
var reverseList = function (head) {
  let current = head;
  let prev = null;

  while (current) {
    let tmp = current.next;
    prev = current;
    current.next = previous;
    current = temp;
  }

  return prev;
};

// Input: head = [1,2,3,4,5] -> Output: [5,4,3,2,1]
```
